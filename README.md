WinXplorer is a Windows Explorer recreation (specifically, the Vista explorer) based in QtQuick, which aims to also add some useful features. Works best with [VistaThemePlasma](https://gitgud.io/catpswin56/vistathemeplasma).

## Installation 

1. Run ``chmod +x install.sh && ./install.sh``
2. The program will be installed as ``winxplorer``
